WhisperX

This modules provides fast automatic speech recognition (70x realtime with large-v2) with word-level timestamps and speaker diarization.

Options

  • model: string, identifier of the model to choose:

    • tiny

    • base

    • small

    • medium

    • large

    • large-v2

  • alignment_mode: string, alignment method to use

    • raw Segments as identified by Whisper

    • segment Improved segmentation using separate alignment model. Roughly equivalent to sentence alignment.

    • word Improved segmentation using separate alignment model. Equivalent to word alignment.

Example payload

payload = {
    'trainerFilePath': 'models\\trainer\\free\\transcript\\whisperx\\whisperx_transcript.trainer',
    'server': '127.0.0.1',
    'username': 'nova-user',
    'password': 'nova-password',
    'database': 'test-db',
    'sessions': 'session_1;session_2',
    'scheme': 'transcript',
    'roles': 'speaker_1;speaker_2',
    'annotator': 'whisper',
    'streamName': 'audio',
    'schemeType': 'FREE',
    'optStr': 'model=base;alignment_mode=word'
}

import requests

url = 'http://127.0.0.1:53770/predict'
headers = {'Content-type': 'application/x-www-form-urlencoded'}
requests.post(url, headers=headers, data=payload)
